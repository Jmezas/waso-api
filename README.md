## Installation

### Clonar el repositorio

```bash
$ git clone https://github.com/real-soft/waso-api.git waso-api
```

### Navegar hacia la carpeta del proyecto

```bash
$ cd waso-api/
```

### Instalación de las dependencias

```bash
$ npm install
```

## Ejecución de la aplicación

```bash
# Modo desarrollo
$ npm run start

# Modo escucha
$ npm run start:dev

# Modo producción
$ npm run start:prod
```
